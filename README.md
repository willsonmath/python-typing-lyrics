# Python Typing Lyrics

My first Python project, built on my Android phone using Pydroid 3 while learning Python.

## Description

This program creates a typing animation for song lyrics in the terminal. Instead of setting the typing speed for each character, you only need to specify:

- The duration of each lyric line.
- The delay before the next line appears.

The program automatically calculates the typing speed based on the length of the text.

## Features

- Typing animation effect
- Adjustable duration for each line
- Adjustable delay between lines
- Easy to customize for any song lyrics

## Example

```python
ketik("Willson Andre Simamora", 5, 2)
ketik("STEI-R ITB 26", 3, 0)
```
